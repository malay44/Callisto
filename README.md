## How to run the app

_Make sure to run the `mongod` service before running the app locally._

`npm i`

mkdir .env as:-
  - DB_STRING=
  - SECRET=
  - GOOGLE_CLIENT_ID=
  - GOOGLE_CLIENT_SECRET=
  - MAIL_API_KEY= 
 Also have to change domain name to localhost:PORT
